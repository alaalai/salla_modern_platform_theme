# ثيم المنصة العصرية - Modern Platform Theme for Salla

## نظرة عامة

**ثيم المنصة العصرية** هو ثيم احترافي وحديث لمنصة سلة، مصمم خصيصاً للمتاجر المتخصصة في بيع المنتجات الرقمية والخدمات الإبداعية. الثيم يجمع بين التصميم الجذاب والأداء العالي والتوافقية الكاملة مع معايير سلة.

---

## الميزات الرئيسية

### 1. تصميم حديث واحترافي
- نظام ألوان متناسق: أزرق داكن وأخضر ليموني
- طباعة عصرية باستخدام Poppins و Inter
- واجهة نظيفة وسهلة الاستخدام

### 2. تصميم مستجيب (Responsive Design)
- يعمل بسلاسة على جميع الأجهزة (الهاتف، الجهاز اللوحي، سطح المكتب)
- تخطيط مرن يتكيف مع حجم الشاشة

### 3. مكونات تفاعلية
- **Header:** رأس صفحة مع شريط إعلانات وقائمة ملاحة
- **Footer:** تذييل شامل مع روابط وسائل التواصل والاشتراك البريدي
- **ProductCard:** بطاقات منتجات مع صور وتقييمات وأسعار مخفضة
- **Testimonials:** قسم شهادات العملاء مع نظام التنقل

### 4. صور عالية الجودة
- صور خلفية مُنشأة بالذكاء الاصطناعي
- تصميم احترافي يعكس جودة الثيم

### 5. أداء عالي
- تحميل سريع للصفحات
- تحسين محركات البحث (SEO)
- دعم الأجهزة القديمة

## التثبيت والتشغيل

### المتطلبات
- Node.js 18+ و npm أو pnpm
- معرفة أساسية بـ React و Tailwind CSS

### خطوات التثبيت

1. **استنساخ المشروع:**
```bash
git clone <repository-url>
cd salla_modern_platform_theme
```

2. **تثبيت الحزم:**
```bash
pnpm install
```

3. **تشغيل خادم التطوير:**
```bash
pnpm dev
```

4. **فتح المتصفح:**
اذهب إلى `http://localhost:3000`

## التخصيص والتطوير

### تغيير الألوان

عدّل ملف `client/src/index.css` وغيّر قيم الألوان في قسم `:root`:

```css
:root {
  --color-dark-blue: #0F172A;      /* غيّر هذا */
  --color-lime-green: #84CC16;     /* أو هذا */
  --color-light-gray: #F3F4F6;
  /* ... */
}
```

### إضافة صفحات جديدة

1. أنشئ ملف جديد في `client/src/pages/`
2. أضف المسار في `client/src/App.tsx`:

```tsx
import NewPage from "@/pages/NewPage";

function Router() {
  return (
    <Switch>
      <Route path="/new-page" component={NewPage} />
      {/* ... */}
    </Switch>
  );
}
```

### تعديل المكونات

جميع المكونات موجودة في `client/src/components/` وجاهزة للتعديل والتطوير.
